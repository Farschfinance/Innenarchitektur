⚙️ Supabase-Verbindung einrichten

Bearbeite diese HTML-Datei und trage deine Supabase-Zugangsdaten ein. Du findest sie in deinem Supabase-Projekt unter Settings → API.

1. Supabase-Projekt erstellen
Gehe auf supabase.com, erstelle ein kostenloses Projekt.
2. Datenbank-Tabellen anlegen
Führe das SQL-Schema unter SQL Editor in Supabase aus. Das Schema findest du als Kommentar am Ende dieser Datei.
3. Zugangsdaten eintragen
Öffne diese Datei in einem Texteditor und ersetze DEINE_SUPABASE_URL und DEIN_SUPABASE_ANON_KEY mit deinen echten Werten.
4. Auf GitHub Pages hosten
Repository erstellen → Datei hochladen → Settings → Pages → Branch: main → Speichern. Fertig!
